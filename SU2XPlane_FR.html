X-Plane plugin pour SketchUp

Vue d'ensemble

SketchUp est un logiciel gratuit, éditeur d'objets 3D pour Windows et MacOS.

Ce plugin SketchUp importe et exporte les modèles vers X-Plane v7 à v10 au format .obj.

Installation

Copiez les fichiers extraits de l'archive .zip vers le dossier Plugins de SketchUp:

  • Windows C:\Program Files (x86)\Google\Google SketchUp n\Plugins ou, si ce dossier n'existe pas,
    C:\Programmes\Google\Google SketchUp n\Plugins
  • Mac OS: /Library/Application Support/Google SketchUp n/SketchUp/plugins

(Re) démarrage de SketchUp.

Vous pouvez maintenant supprimer le fichier .zip que vous avez téléchargé. Ce fichier d'aide peut être consulté en cliquant sur Aide → X-Plane dans SketchUp.

Création d'objets décors X-Plane

Avant de commencer, vous pouvez obtenir une photo de la zone que vous êtes sur le point de modèlisé:

  • Choisissez Fenêtre → Infos sur le modèle → Géoposition → Ajouter un emplacement…

Créez votre modèle dans SketchUp avec des rectangles, des cercles et des arcs. Enregistrez le fichier SketchUp .skp dans le dossier des avions ou de paysage où vous voulez que le fichier X-Plane .obj se retrouve.

Dans une application d'édition d'image, créer un fichier de texture qui ressemble à un «collage» de toutes les textures que vous avez besoin dans votre modèle. La hauteur et la largeur du fichier texture doit être en puissance de deux (par exemple 128, 256, 512, 1024, 2048) mais pas spécialement carré. Enregistrez le fichier texture en 32 bits ou 24 bits au format PNG (c'est à dire avec ou sans canal «Alpha») dans le dossier avion ou paysage où vous voulez que le fichier X-Plane .obj se retrouve.

Utilisez l'outil Colorier pour appliquer votre texture d'un fichier à chaque face de votre modèle. Utilisez le menu contextuel → Texture → Position de cartographie la partie correcte de ce fichier de texture à chaque face.

Pour placer une texture sur une surface courbe, voir ici un tutoriel. Ou vous pouvez d'abord activer Affichage → Géométrie cachée et ensuite, sélectionnez l'une des faces de la surface à un moment et placer la texture sur chaque face.

Si vous voulez que les deux côtés d'une face soient affichées dans X-Plane, appliquer l'outil Colorier à deux reprises, une fois de chaque côté dela face. Vous pouvez utiliser la Fenêtre → Infos sur l'entité panneau pour voir quelles textures sont actuellement affectées aux deux côtés de la face sélectionnée.

Vous pouvez utiliser Outils → Highlight Untextured pour vous aider à texturer. Cela ne va pas changer la façon dont votre modèle apparaîtra dans X-Plane, mais elle mettra en évidence, dans SketchUp, des problèmes avec votre texture:

  1. Faces non texturés colorées en rouge. Cela vous aide à identifier ces faces.
  2. Pour les faces qui sont texturées sur un seul côté, l'autre côté est caché (régler l'opacité à 0). Cela vous permet de voir si vous avez texturé du mauvais côté toutes les faces. Utilisez le menu contextuel → Inverser les faces sur la face si vous aviez cela.

Attributs

Ce plugin ajoute trois entrées au menu contextuel. Ceux-ci peuvent être utilisés lorsque vous avez une face sélectionnée:

Hard
Contrôle si l'aéronef peut atterrir sur et / ou entrer en collision avec une face dans X-Plane.
Faire faces "hard" est couteux, donc utiliser cette option avec parcimonie. Vertical "hard" des faces ne fonctionnent pas comme on pouvait s'y attendre, si vous voulez faire un bâtiment «collisable» il suffit d'utiliser ce bouton pour le toit mais pas sur les côtés du bâtiment.
 
Ground
Contrôle si la face est rendue avec «polygon offsetting» dans X-Plane.
Utilisez ce sur les faces qui se trouvent à plat sur le sol pour éviter de raclée (Z-buffer) sous la surface d'X-Plane. Ne pas appuyer sur ce bouton pour les autres faces.
 
Alpha
Il s'agit d'une allusion à X-Plane pour que la face soit texturée avec une texture partiellement ou totalement transparente ou translucide.
Utilisez-le sur toutes les faces transparentes ou translucides à travers lesquelles vous voulez voir les autres faces du même modèle. Ne pas appuyer sur ce bouton pour les faces opaques (normale).

Lumières

Utilisez l'outil Texte pour créer des lumières:

  • Tout d'abord, utilisez l'outil Mètre pour créer un «Point de guidage» à l'endroit où vous voulez que la lumière apparaisse.
  • Choisissez Outils → Texte.
  • Sélectionnez le point que vous venez de créer, puis faites glisser votre souris (ce qui crée une flèche pointant vers votre point de repère), puis cliquez dessus.
  • Tapez le nom de l'un des types de lumière documentés ici, c'est-à-dire LIGHT_NAMED, LIGHT_CUSTOM, LIGHT_PARAM, LIGHT_SPILL_CUSTOM, smoke_black ou smoke_white.
  • Tapez également les paramètres requis, sauf pour les paramètres <x> <y> <z>, dont les valeurs seront fournies par le plugin.
    Remarque : Pour les lumières de type LIGHT_NAMED et LIGHT_PARAM le fichier X-Plane/Resources/bitmaps/world/lites/lights.txt répertorie les valeurs utilisables pour le paramètre <name>.

Exportation vers X-Plane

Si vous ne l'avez pas encore fait, choisissez Fichier → Enregistrer sous… et enregistrez le fichier .skp dans le dossier avion ou le dossier paysage où vous voulez que le fichier X-Plane .obj se retrouve.

Si vous avez chargé une photo du terrain de Google Earth, il vous faut le cacher avant de les exporter:

  • Choisissez Fenêtre → Calques.
  • Dans le panneau Calques veiller à ce que «Google Earth terrain» et «Google Earth Snapshot» soient décochés.
  • Si vous avez chargé la photo du terrain avant de commencer la modélisation, vous pouvez lire les coordonnées où votre modèle doit être placé dans X-Plane à partir du panneau Fenêtre → Infos sur le modèle → Géoposition.

Ensuite:

  • Choisissez Fichier → Export X-Plane Object

Vous pouvez avoir les avertissements suivants:

Warning: You used multiple texture files. Using file name.
Vous avez utilisé plus d'un fichier de texture pour les faces de votre modèle.
X-Plane vous oblige à n'utiliser qu'un seul fichier de texture pour l'ensemble des faces de votre modèle.
 
Warning: n faces are untextured.
Vous n'avez pas appliqué de texture à l'ensemble des faces de votre modèle SketchUp.
X-Plane sera de la couleur des faces non texturées avec le pixel en bas à gauche de votre fichier de texture (ou gris si vous n'avez pas appliqué une texture à l'une des faces).

Si vous voyez l'un de ces avertissements, X-Plane ne sera probablement pas capable d'afficher votre modèle comme vous le souhaitiez.

Limites

Construisez votre modèle SketchUp avec des rectangles, des cercles, des arcs, les Composants et Groupes seulement - tous les autres types de SketchUp ne seront pas exportés.

Ce plugin ne vous permet pas de créer des primitives spécifiques dans X-Plane, comme:

  • Les animations
  • Des attributs: cockpit par exemple, matériaux, etc
  • Des niveaux de détail
  • Des lignes.
