X-Plane plugin for SketchUp

Overview

SketchUp is a free (as in beer) 3D object editor for Windows and MacOS.

This SketchUp plugin exports models to X-Plane v7 or v8 .obj format.

Installation

Copy the two files SU2Xplane.html and SU2Xplane.rb into:

Google SketchUp on Windows: C:\Program Files\Google\Google SketchUp\Plugins
SketchUp Pro 5 on Windows: C:\Program Files\@Last Software\SketchUp 5\Plugins
Mac OS X: ~/Library/Application Support/SketchUp/Plugins

(Re)start SketchUp. This help file can now be viewed by choosing Help » X-Plane in SketchUp.

You can now delete the .zip file that you downloaded.

Creating X-Plane scenery objects

Create your model in SketchUp out of Rectangles and Circles. (Any Arcs will be ignored).

In an image editor application create a single texture file that contains all of the textures that you need in your model.

Save the SketchUp .skp file and the texture file in the aircraft or scenery folder where you want the X-Plane .obj file to end up.

Use the Paint Bucket tool to apply the same texture file to every surface in your model. Use Context menu » Texture » Position to map the correct portion of this texture file to each surface. If you want both sides of a surface to be displayed in X-Plane then apply the Paint Bucket tool twice; once on each side of the surface.

Exporting to X-Plane

If you have not already done so, choose File » Save As... and save the .skp file in the aircraft or scenery folder where you want the X-Plane .obj file to end up.

Then choose one of:

File » Export X-Plane v7 Object
v7 objects can be used in X-Plane v7 or v8. However curved surfaces in SketchUp will appear "steppy" in X-Plane.
File » Export X-Plane v8 Object
v8 objects can only be used in X-Plane v8.20 and later. Curved surfaces will be displayed correctly.

The object is created in the same folder and with the same name as the current SketchUp file, but with a .obj extension.

You may see the following warnings:

Warning: You used multiple texture files. Using file name.
You used more than one texture file to texture the surfaces in your model. X-Plane requires that you use only one texture file for all of your surfaces.
Warning: n of those triangles are untextured.
Some of the surfaces in your SketchUp model didn't have a texture applied to them. X-Plane requires that all of your surfaces are textured.

If you see either warning then X-Plane probably won't display your model correctly.

Limitations

Construct your SketchUp model out of Rectangles, Circles, Components and Groups only - any other SketchUp types (including Arcs) are not exported.

This plugin does not allow you to create advanced X-Plane scenery primitives, including:

  • Animations
  • Hard surfaces
  • Lights
  • Lines
  • Smoke
