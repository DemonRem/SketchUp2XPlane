X-Plane plugin for SketchUp

Overview

SketchUp is a free (as in beer) 3D object editor for Windows and MacOS.

This SketchUp plugin exports models to X-Plane v7 or v8 .obj format.

Installation

Copy the two files SU2Xplane.html and SU2Xplane.rb into your SketchUp Plugins folder:

Google SketchUp on Windows: C:\Program Files\Google\Google SketchUp\Plugins
Google SketchUp on MacOS: /Library/Application Support/Google SketchUp/Plugins
SketchUp Pro 5 on Windows: C:\Program Files\@Last Software\SketchUp 5\Plugins
SketchUp Pro 5 on MacOS: /Users/you/Library/Application Support/SketchUp 5/Plugins

(Re)start SketchUp.

You can now delete the .zip file that you downloaded. This help file can be viewed by choosing Help » X-Plane in SketchUp.

Creating X-Plane scenery objects

Create your model in SketchUp out of Rectangles and Circles (any Arcs will be ignored). Save the SketchUp .skp file in the aircraft or scenery folder where you want the X-Plane .obj file to end up.

In an image editor application create one texture file that is like a "collage" of all of the textures that you need in your model. The height and width of the texture file must be powers of two (eg 128, 256, 512, 1024) but it doesn't have to be square. See .../Custom Scenery/KSBD Demo Area/KSBD_example.png for a basic example.

Save the texture file in 24bit or 32bit PNG format (ie with or without an "Alpha" channel) in the aircraft or scenery folder where you want the X-Plane .obj file to end up.

Use the Paint Bucket tool to apply your one texture file to every surface in your model. Use Context menu » Texture » Position to map the correct portion of this texture file to each surface (see here for a tutorial).

To position a texture on a curved surface you first have to enable View » Hidden Geometry. Then select one facet of the surface at a time and position the texture on each facet.

If you want both sides ("faces") of a surface to be displayed in X-Plane then apply the Paint Bucket tool twice; once on each face of the surface. You can use the Window » Entity Info panel to see which texture(s) are currently assigned to the two faces of the currently selected surface.

You can use Tools » Highlight Untextured to help you with texturing. This doesn't change how your model will appear in X-Plane, but it does highlight any problems with your texturing in SketchUp:

  1. Untextured surfaces are coloured red. This helps you to identify these surfaces.
  2. For surfaces that are textured on just one face, the other face is hidden (by setting the opacity to 0). This helps you to see if you've textured the wrong face of any surfaces. Use Context menu » Reverse Faces on the surface if you have.

Exporting to X-Plane

If you have not already done so, choose File » Save As... and save the .skp file in the aircraft or scenery folder where you want the X-Plane .obj file to end up.

Then choose one of:

File » Export X-Plane v7 Object
v7 objects can be used in X-Plane v7 or v8.
Curved surfaces will appear "steppy" in X-Plane.
 
File » Export X-Plane v8 Object
v8 objects can only be used in X-Plane v8.16 and later.
X-Plane will display curved surfaces correctly.

The object is created in the same folder and with the same name as the current SketchUp file, but with a .obj extension.

You may see the following warnings:

Warning: You used multiple texture files. Using file name.
You used more than one texture file to texture the surfaces in your model.
X-Plane requires that you use only one texture file for all of your surfaces.
 
Warning: n surfaces are untextured.
You have not applied a texture to all of the surfaces in your SketchUp model.
X-Plane will colour the untextured surfaces with the bottom left pixel from your texture file (or grey if you have not applied a texture to any of the surfaces).

If you see either warning then X-Plane probably won't display your model as you intended.

Limitations

Construct your SketchUp model out of Rectangles, Circles, Components and Groups only - any other SketchUp types (including Arcs) are not exported.

This plugin does not allow you to create advanced X-Plane scenery primitives, including:

  • Animation
  • Attributes: eg hard surfaces, poly_os, cockpit, materials etc
  • Level Of Detail
  • Lights
  • Lines
  • Smoke
