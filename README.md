# SketchUp2XPlane
## XPlane plugin for SketchUp

This [SketchUp](http://www.sketchup.com/) plugin imports and exports models in [X-Plane](http://www.x-plane.com/) v8/v9/v10 `.obj` format.
Runs on Windows XP or later and Mac OS 10.5 or later. Requires SketchUp 7 or later.

The latest released version of this plugin is available packaged as a SketchUp `.rbz` extension from http://marginal.org.uk/x-planescenery/tools.html .

Please refer to the documentation, which is available after installation by choosing _Help â†’ X-Plane_ in SketchUp.
